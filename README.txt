The code is written in Arduino IDE.


To install i2cdev.h library :
1. visit the link - https://github.com/jrowberg/i2cdevlib
2. And install the i2cdev.h library into the system into desired location.
3. In the Arduino IDE, go to Sketch -> Include Library -> Add zip library -> library location


To install MPU6050.h
1. Install MPU6050.h from the link - https://drive.google.com/file/d/1LiY4DTqVo6vXTd-re3vsjldu3bjaU3Wd/view
2. In the Arduino IDE, go to Sketch -> Include Library -> Add zip library -> library location


To install Mouse.h library :
